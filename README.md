# which_app

学校でFlutterを使ってみようという授業の際に作ったアプリです。

このアプリは普段生活している中で「コレとコレだったらみんなはどっち選ぶんだろう？」という素朴な疑問を気軽に投稿し、投票し合えます。
（例）たけのこの里 vs きのこの山　など...
